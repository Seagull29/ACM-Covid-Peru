# ACM-Covid-Peru+
Elaborado Por: *Seagull29 & Dyriss*
Segunda Aplicación de conceptos anteriores, con el uso de una base de datos referente a COVID-19 en Perú. 
El proyecto utiliza funciones de agregar, editar y eliminar datos de una base que contiene casos por región, 
incluyendo Pruebas, Casos Positivos, Pruebas Rapidas y Letalidad. (MINSA)
Los datos son mostrados en graficos con la función mostrar grafico en la segunda pestaña.
